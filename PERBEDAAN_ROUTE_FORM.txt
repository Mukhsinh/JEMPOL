========================================
PERBEDAAN ROUTE FORM TIKET INTERNAL
========================================

Ada 2 route berbeda untuk form tiket internal:

1. /form/internal (DirectInternalTicketForm)
   ✓ TANPA SIDEBAR
   ✓ TANPA LOGIN
   ✓ Tampilan mobile-first, fullscreen
   ✓ Untuk QR Code di lokasi fisik
   ✓ File: frontend/src/pages/public/DirectInternalTicketForm.tsx

2. /tickets/create/internal (CreateInternalTicket)
   ✓ DENGAN SIDEBAR
   ✓ PERLU LOGIN
   ✓ Tampilan dashboard admin
   ✓ Untuk admin yang sudah login
   ✓ File: frontend/src/pages/tickets/CreateInternalTicket.tsx

========================================
KONFIGURASI QR CODE MANAGEMENT
========================================

Di halaman /tickets/qr-management:
- Saat membuat QR Code baru
- Pilih "Arahkan ke" = "Form Tiket Internal"
- Link yang dihasilkan HARUS: /form/internal
- Link yang dihasilkan JANGAN: /tickets/create/internal

========================================
CARA VERIFIKASI
========================================

1. Buka halaman QR Management
2. Klik "Buka Link" pada QR Code
3. Periksa URL di browser:
   - BENAR: http://localhost:3003/form/internal?qr=...
   - SALAH: http://localhost:3003/tickets/create/internal?...

4. Periksa tampilan halaman:
   - BENAR: Tidak ada sidebar, fullscreen form
   - SALAH: Ada sidebar di kiri

========================================
SOLUSI JIKA MASIH SALAH
========================================

1. Clear browser cache (Ctrl+Shift+Delete)
2. Restart aplikasi frontend
3. Edit QR Code yang sudah ada:
   - Ubah "Arahkan ke" ke pilihan lain
   - Simpan
   - Ubah kembali ke "Form Tiket Internal"
   - Simpan lagi
4. Test link yang baru dihasilkan
